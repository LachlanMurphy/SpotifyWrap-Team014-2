Recommendations feature: Searching for a song will give 5 song recommendations as well as the song being searched for. Searching for an artist will give artist recommendations as well as the artist being searched for.
Like/Favorite feature: User will be able to favorite songs that they search and songs will be put into a database 
Search feature: Search for songs from Spotify's database
Login: Logs in user
Registration: Registers users 
Logout: Logs out users 
Nav bar: Navigating through pages
Profile Page: User can view their user information
Favorites Page: User can view their favorite songs
